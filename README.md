# Matemática Aplicada

Autor: Dr. Pedro Alencar
Contato: pedro.alencar@campus.tu-berlin.de

Neste repositório, você encontrará materiais e recursos relacionados à matemática aplicada, incluindo slides das aulas, referências bibliográficas, artigos científicos e scripts utilizados nas aulas (com e sem resolução). 

O conteudo é referente as aulas 9 a 12:

- Aula 9: Equações Diferenciais Ordinárias
- Aula 10: Equações não lineares e otimização
- Aula 11: Séries temporais
- Aula 12: Sensibilidade e incerteza

## Estrutura do Repositório

- `Aula9`: Slides9.pdf e pasta `ref` com referências bibliográficas e artigos científicos.
- `Aula10`: Slides10.pdf, pasta `ref` com referências bibliográficas e artigos científicos.
- `Aula11`: Slides11.pdf, pasta `ref` com referências bibliográficas e artigos científicos.
- `Aula12`: Slides12.pdf, pasta `ref` com referências bibliográficas e artigos científicos.
- `code`: Scripts utilizados em cada aula `Aula*.R`, bem como scripts com a exemplos e exercícios resolvidos `Aula*_solved.R`.

## Como Usar Este Repositório

1. Clone o repositório em sua máquina local.
2. Navegue até a pasta da aula desejada.
3. Abra os arquivos PDF para visualizar os slides.
4. Consulte a pasta `ref` para obter referências adicionais.
5. Execute os scripts R na pasta `code` para reproduzir os exemplos e exercícios.
